関心分野・学んでいること

現在はソフトウェアとハードウェアの両方を学習し、実際に実験で動作を確認したりしています。以下の分野を特に学んでいます。

  • 組込みシステム：
    ハードのものをソフトで動かす仕組みなどを学んでいます。
  • 画像処理：
    画像データから対象物を取り出す仕組みや技術を学んでいます。
  • 統計：
    実際の統計の計算を手計算で解くだけでなくAIを駆使しながら適切な計算方法や手段を学んでいます。

授業での成果物・プロジェクト

1. Javaプログラミングによる暗算ゲームの作成

Java

概要： 提示された数字を暗算で素因数分解を行い、10問解いて正答率とかかった時間が表示される。個人でタイムアタックや暗算のスピード向上が狙える。

工夫点： ただの計算プログラミにならないために時間の表示や正答率の計算、コード内部だと正解の判定の出し方を工夫している。。

2. パワーポイントを使用した発表

PowerPoint

概要： 実際の授業で自分が大学で学びたい内容や興味のあるものを選択し、スライドを作成し発表を行った。

工夫点： スライドの内容に画像をいれて理解しやすくしたり、文章を多用しないことで見やすいスライドづくりを意識した。
